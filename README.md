# JPush API Python Client

JPush REST APIs 的 Python 语言版本开发包。由 JPush 官方维护。

## 安装

通过 pip 命令：

    $ sudo pip install jpush

或者使用 easy_install：

    $ sudo easy_install jpush

或者直接从源代码安装：

    $ sudo python setup.py install
