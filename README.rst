===================
JPush Python Client
===================

A Python client for the JPush REST APIs

------------
Installation
------------
To install jpush-api-python-client, simply:

.. code-block:: sh

    $ sudo pip install jpush

or alternatively install via easy_install:

.. code-block:: sh

    $ sudo easy_install jpush


or from source:

.. code-block:: sh

    $ sudo python setup.py install
